Welcome in Unification project which unites two kata tasks and battle ships game as far.

The project was done to train binding different modules together.

Anagrams kata is supposed to search an anagram from built-in word list.
Diversion converts number to binary value and checks whether there are adjecent 1 in each chain.
User decides about length of the chains.

Katas are taken from https://github.com/gamontal/awesome-katas
